
# generator-webpack-vm 
一个基于Webpack 的 移动端Vue 项目， 这个项目依赖于 yeoman 和 generator 进行运行 

首先先进行安装 

## 安装
请先安装 yeoman 再安装 generator-webpack-vm , 之后就可以进行生成这个脚手架 
```bash
npm install -g yo
npm install -g generator-webpack-vm
```

真正的进行生成一个项目
```bash
yo webpack-vm
```
